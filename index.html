About me

My name is Diego Llanes Pasos. I'm a Mechatronics Engineer graduated from the Autonomous University of Yucatan.
The subjects I was most passionate about in university were those related to programming and electronics. Because of my degree competencies, I had a competent background in programming, I build projects using C and Java but I had no experience in Web Development. For this reason, I decided to study Front-end Web Development through online learning plattforms. I started from the basics: HTML, CSS, Javascript and finally I pick React because I considered it was the framework with the most potential.
I’m very curious about new technology and new ways of doing things. I love to build fun projects and read about programming and I try to translate that knowledge as much as possible into the work I do. When I’m not coding, you can find me playing videogames.
